What is this site?

This site showcases forensic anthropology data science projects, primarily using Julia and R programming languages. All source code is provided 'as is' and is available as open source under the GPL version 2 or MIT licenses. Integrated code follows the licensing terms of its original sources, detailed in the root of the source code files.



What's new

OsteoSort is undergoing a complete redevelopment in Julia. Version 1.4.0 is expected to be released in early 2025.

StatureID is a forthcoming application, set to launch in Fall 2024, designed to calculate stature point estimates and prediction intervals through clustering analysis.



The Open Source Standard

This website and all its projects adhere to an open source standard. This entails full and open disclosure of all source code to facilitate future auditing and traceability, comprehensive version control and archiving, and complete transparency in tracking identified bugs or errors. For a detailed list of open source licenses, visit https://opensource.org/licenses.
